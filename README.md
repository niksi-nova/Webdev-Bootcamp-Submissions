# Coding Club WebDev Bootcamp Submissions!
🚀 Bootcamp Assignment Submissions Repository

Welcome! This repository is the official submission portal for all bootcamp assignments.

Please read the instructions carefully before submitting your work. Following the correct process ensures smooth reviews and avoids rejection of your submission.

---

📌 Submission Workflow

All submissions must follow this process:

1. Fork this repository to your GitHub account
2. Clone your forked repository to your local machine
3. Create your submission inside the "submissions/" folder (structure explained below)
4. Commit and push your changes to your fork
5. Raise a Pull Request (PR) to this repository

⚠️ Direct pushes to this repository are not allowed

---

📁 Folder Structure

You must strictly follow the structure below:

submissions/
  yourname_rollnumber/
    assignment-1/
      (your files here)
    assignment-2/
      (your files here)

✅ Example:

submissions/
  anika_23CS101/
    assignment-1/
      solution.pdf
    assignment-2/
      code.py

---

📝 Naming Guidelines

- Folder name format: "yourname_rollnumber"
- Use lowercase and avoid spaces
- Example: "rahul_21ME045"

---

📤 Pull Request Guidelines

When creating your PR:

- Title format:
  "Assignment X Submission - Your Name"

- Description must include:
  
  - Your Name
  - Roll Number
  - Assignment Number

Example:

Name: Anika  
Roll Number: 23CS101  
Assignment: 1

---

⚠️ Important Rules

- ❌ Do not modify other students' folders

- ❌ Do not change the repository structure

- ❌ Do not submit outside the "submissions/" directory

- ❌ Do not create multiple PRs for the same assignment

- ✅ Ensure your files are properly named and organized

- ✅ Double-check before submitting

---

📌 Evaluation Notes

- Only correctly structured submissions will be reviewed
- Improper PRs may be rejected without feedback
- Late or incorrect submissions may not be considered

---

💡 Need Help?

If you face any issues:

- Recheck the instructions above
- Ask in the communication channel provided by mentors
